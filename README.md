## 📖 About The Project

<div align="center">
<img src="https://github.com/Harshal350/Pokemon-Battle/blob/main/ASSETS/Banner.png" alt="About Banner" width="700" />
</div>

<br>

**Pokémon Battle Game** brings the iconic Pokémon turn-based battle experience straight to your browser — no downloads, no installs, just pure nostalgic fun! Choose your Pokémon, select your attacks, watch the animations unfold, and outsmart your opponent to victory.

Whether you're a **seasoned developer** looking to explore game logic, or a **Pokémon fan** wanting to relive the classics, this project has something exciting for everyone. Built entirely with **HTML, CSS, and JavaScript** — no frameworks, no libraries — just raw web magic! ✨

```
🔴 Choose Pokémon  ──►  ⚔️ Select Attack  ──►  💥 Watch Animation  ──►  🏆 Win or Lose!
```

## ✨ Features

## 🚀 Getting Started

### Prerequisites

Before you begin, make sure you have the following installed:

- ✅ A modern web browser (Chrome, Firefox, Edge, Safari)
- ✅ [Git](https://git-scm.com/) *(optional — for cloning)*
- ✅ A code editor like [VS Code](https://code.visualstudio.com/) *(optional — for development)*

---

### 📥 Installation

**Method 1 — Clone the Repository** *(Recommended)*

```bash
# 1. Clone the repo
git clone https://github.com/Harshal350/Pokemon-Battle.git

# 2. Navigate into the project directory
cd Pokemon-Battle

# 3. Open the game in your browser
open index.html
# ── OR ──
# Simply drag index.html into any browser window!
```

**Method 2 — Download ZIP**

```
1. Click the green "Code" button at the top of this page
2. Select "Download ZIP"
3. Extract the ZIP to your desired folder
4. Open index.html in your browser — done! 🎉
```

**Method 3 — Live Demo**

## 🎮 How To Play

<div align="center">
<img src="https://github.com/Harshal350/Pokemon-Battle/blob/main/ASSETS/1st%20Screen.png" alt="How To Play" width="700" />
</div>

<br>

### 🕹️ Step-by-Step Guide

```
┌─────────────────────────────────────────────────────────┐
│  STEP 1  │  Open the game in your browser               │
│  STEP 2  │  Choose your Pokémon from the selection      │
│  STEP 3  │  Pick an attack from your move set           │
│  STEP 4  │  Watch the attack animation unfold!          │
│  STEP 5  │  The enemy AI retaliates automatically       │
│  STEP 6  │  Reduce the enemy HP to 0 to WIN 🏆          │
│  STEP 7  │  Don't let YOUR HP hit 0 or you LOSE 💀      │
└─────────────────────────────────────────────────────────┘
```

### ⌨️ Controls & Keyboard Shortcuts

| Key / Input | Action |
|:---:|:---|
| `Mouse Click` | Select Pokémon / Choose Attack |
| `Enter` | Confirm selection |
| `Escape` | Pause / Return to menu |
| `WASD` | SELECT OPTIONS FRO PLAYER 1 |
| `ARROW KEYS ` | SELECT OPTION FOR PLAYER 2 |
| `S` | Toggle sound effects 🔊 |
| `F` | Toggle fullscreen 🖥️ |

## 📁 Folder Structure

```
Pokemon-Battle/
│
├── 📄 index.html              # Main entry point

│   ├── 🎨 style.css           # Main stylesheet

│   ├── ⚙️  game.js             # Core game logic & state
│   ├── ⚙️  battle.js           # Battle system & turn engine

│   ├── ⚙️  animations.js       # Animation controller

│   │   └── 🖼️  ui/             # Buttons, icons, UI elements

│       └── 🔊 sfx/             # UI & game sound effects
│
└── 📂 data/
    └── 📄 pokemon-data.json   # Pokémon stats, moves & types
```
